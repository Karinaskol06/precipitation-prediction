## Про проєкт

Проєкт демонструє створення ML-застосунку від отримання даних до візуалізації прогнозів. Він навчається на історичних метеоданих і будує прогноз на основі 13 різних ознак (погода сьогодні, тенденції за 3 дні, сезонність).

### Що вміє застосунок?
- Автоматично завантажувати дані з Open-Meteo API
- Візуалізувати історичні дані (опади, температура)
- Навчати 1-7 різних ML-моделей (окремо для кожного дня)
- Оцінювати якість моделей (accuracy, precision, recall, f1)
- Прогнозувати опади на 1-7 днів вперед
- Показувати результати у веб-інтерфейсі

## Моделі машинного навчання

### Два алгоритми на вибір:

**1. Random Forest Classifier** 
- Будує 200 різних дерев рішень на випадкових підмножинах даних
- Кожне дерево створює прості правила: "якщо температура > 20 і вітер < 10 - сухо"
- Фінальний прогноз визначається голосуванням більшості дерев

**2. Logistic Regression**
- Будує одну математичну формулу виду: ймовірність_дощу = 1 / (1 + e^-(w1*temp + w2*wind + w3*precip + ...))
- Коефіцієнти w1, w2, w3 показують вагу кожної ознаки
- Тільки лінійні залежності - не бачить складних взаємодій

### Ознаки для прогнозу:
```python
self.feature_names = [
    # Поточна погода (7)
    'temp_max_today',           # макс температура сьогодні
    'temp_min_today',           # мін температура сьогодні
    'wind_speed_today',         # швидкість вітру сьогодні
    'precip_sum_today',         # опади сьогодні
    'rain_sum_today',           # дощ сьогодні
    'radiation_today',          # сонячна радіація
    'sunshine_hours_today',     # сонячні години
    
    # Тенденції (4)
    'precip_mean_3d',           # сер. опади за 3 дні
    'temp_max_mean_3d',         # сер. температура за 3 дні
    'rainy_days_3d',            # днів з дощем (з 3)
    'wind_speed_mean_3d',       # сер. вітер за 3 дні
    
    # Сезонність (2)
    'month',                     # місяць
    'day_of_year'                # день року
]
```
